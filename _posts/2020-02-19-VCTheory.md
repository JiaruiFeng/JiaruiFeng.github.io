Generalization is a key issue in learning. As we defined in learning theory, we can use *generalization error* as the discrepancy between $E_{in}$ and $E_{out}$, use hoeffding inequality:


$$
\mathbb{P}\left[\left|E_{\mathrm{in}}(g)-E_{\mathrm{out}}(g)\right|>\epsilon\right] \leq 2 M e^{-2 \epsilon^{2} N}
$$


for any $\epsilon>0$, we can reshape above inequality as follows. Pick a tolerance level $\delta$, with the probability at least $1-\delta$:


$$
E_{\mathrm{out}}(g) \leq E_{\mathrm{in}}(g)+\sqrt{\frac{1}{2 N} \ln \frac{2 M}{\delta}}
$$


However, when $M \rightarrow \infty$, the bound goes to infinity as well. and this is the case the majority of algorithms. In order to study generalization in such models, we need to replace this infinity $M$ with a finite item. Recall the bad event $B_{m}$ of $h_{m}$, there are always overlapping between every $B_{i}$, Thus, we actually over-estimated the probability using union bound.

#### Effective number of Hypothesis

Instead of using number of hypothesis, we count "effective" number of hypothesis. First we introduce two definition.

**Dichotomy**: a set of possible prediction combinations $$h \in {H}$$ can induce on $$\vec{x}_{1}, \ldots, \vec{x}_{N}$$. Characterized by both $H$ and $N$ data points.


$$
H\left(\vec{x}_{1}, \ldots \vec{x}_{N}\right)=\left\{h\left(\vec{x}_{1}\right), \ldots, h\left(\vec{x}_{N}\right) | h \in H\right\}
$$


**Growth Function**:Largest number of dichotomies that $H$ can induce across all possible data sets of size $N$.


$$
m_{H}(N)=\max _{\left(\vec{x}_{1}, \ldots, \vec{x}_{N}\right)}\left|H\left(\vec{x}_{1}, \ldots, \vec{x}_{N}\right)\right|
$$


In other words, $m_{H}(N)$ is the maximum number of dichotomies that can be generated by $\mathcal{H}$ on any $N$ points. Here are some examples:

![image-20200128170636483](/assets/images/image-20200128170636483.png)

Now, we want to use Growth Function to replace $M$, turns out we have **VC Generalization Bound** theorem: With the probability at least $1-\delta$,


$$
E_{o u t}(g) \leq E_{i n}(g)+\sqrt{\frac{8}{N} \ln \frac{4 m_{H}(2 N)}{\delta}}
$$



#### How to Bound Growth Function

Turns out define $m_{H}(N)$ for different hypothesis sets is diffcult, we need find a way to bound growth function. We define follow definitions:

**Shatter**: if the number of dichotomies  for $$\vec{x}_{1}, \ldots, \vec{x}_{N}$$ is $$2^N$$, we say $$H$$ can shatter $$\vec{x}_{1}, \ldots, \vec{x}_{N}$$.

**Break Point**: $k$ is a break point for $H$ if no data set of size $k$ can be shattered by $H$.

With these definitions, we have key result:

1. if $k$ is a break point for $H$,then we have $m_{H}(N) \leq \sum_{i=0}^{k-1}\left(\begin{array}{l}
   {N} \\
   {i}
   \end{array}\right)$ . That is $m_{H}(N) \leq N^{k-1}+1$, So $m_{H}(N)$ is polynomial in $N$, in particular $m_{H}(N)=O\left(N^{k-1}\right)$. (skip proo)
2. If there is no break point for $H$, $m_{H}(N)=2^N$ .



Now, we can transform **VC Generalization Bound** as: with the probability at least $1-\delta$,


$$
E_{o u t}(g) \leq E_{i n}(g)+O(\sqrt{(k-1) \frac{\ln (N / \delta)}{N}})
$$



#### VC dimension

The VC dimension of $H$ is the largest $N$ such that $$m_{H}(N)=2^N$$. that is, let $$k^*$$ be the smallest break option for H, then $$d_{vc}(H)=k^*-1$$.

![image-20200211235052363](/assets/images/image-20200211235052363.png)

Here is an example about break points and VC dim

then, play VC dimension on VC bound: with the probability at least $1-\delta$,


$$
E_{o u t}(g) \leq E_{i n}(g)+O(\sqrt{d_{V C} \frac{\log (N / \delta)}{N}})
$$

Above picture illustrate the relationship between hoeffding inequality, union bound and VC bound.(a) is the bad event for a particular hypothesis by appling hoeffding inequality. (b) is bad events for all hypotheses in hypothesis sets. But actually there are lots of bad events overlapped. if we strip all the overlapped events, we will get (c), which is VC bound.

There are two reason why we care about VC theory. First, it establish the "learnability" of infinite $H$. Second, VC bound is loose, but it's roughly equally loose for different learning models.



#### Discussion on the  VC theory

**Complexity of Hypothesis Set**

if we regrad $d_{vc}$ as a feature to characterize the complexity of hypothesis set, with the increase of $d_{vc}$, we can draw a picture based on VC bound:

<img src="/assets/images/image-20200211232602674.png" alt="image-20200211232602674" style="zoom:50%;" />

With the increase of $d_{vc}$, the in-sample error should decrease since we have more chance to find a hypothesis that fit the train data. However, the out-of-sample error would first decrease then increase, since VC bound become larger.

**Sample Complexity**

We define sample complexity as how many data points do we need to achieve generalization error less than $\epsilon$ with probability of $1-\delta$.

With VC bound, we have:


$$
E_{o u t}(g) \leq E_{i n}(g)+\sqrt{\frac{8}{N} \ln \frac{4\left((2 N)^{d_{vc}}+1\right)}{\delta}}
$$


Set $\sqrt{\frac{8}{N} \ln \frac{4\left((2 N)^{d} v c+1\right)}{\delta}} \leq \epsilon$, we get:


$$
N \geq \frac{8}{\epsilon^{2}} \ln \left(\frac{4\left(1+(2 N)^{d_{vc}}\right)}{\delta}\right)
$$


We can see $N \propto 1 / \epsilon^{2}$ and $N=O\left(d_{v c} \ln N\right)$.
